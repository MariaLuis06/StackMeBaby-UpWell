## Descrição

A nossa equipa está a desenvolver uma aplicação que ajuda pessoas com doenças já diagnosticadas a acederem a informação fidedigna sobre cuidados alimentares, cuidados físicos e medicação, usando fontes hospitalares e científicas confiáveis conbatendo assim a desinformação e melhorar a qualidade de vida das pessoas com doenças diagnosticadas
